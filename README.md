# katabumpKeepAlive
katabumpKeepAlive，保活katabump
定时任务：北京时间 09:00 和 21:00 运行

在你的 GitHub 仓库中，依次点击： Settings -> Secrets and variables -> Actions -> New repository secret。
添加以下 Secret：
•	KATA_EMAIL: 你的登录邮箱
•	KATA_PASSWORD: 你的登录密码
•	KATA_SERVER_ID: 你的服务器 ID
•	TG_BOT_TOKEN: (可选) Telegram 机器人 Token
•	TG_USER_ID: (可选) 你的 Telegram 用户 ID
